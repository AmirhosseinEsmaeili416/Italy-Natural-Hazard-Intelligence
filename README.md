# 🇮🇹 Italy Natural Hazard Intelligence

## Project 01 – Flash Flood Intelligence in Liguria

### Overview

Italy faces recurrent flash flood events, especially in Liguria, where steep terrain and intense rainfall create significant natural hazards.

This project investigates how open environmental datasets and geospatial analysis can support flood risk assessment and engineering decision-making.

### Objectives

- Analyze historical rainfall patterns
- Explore hydrological indicators
- Build interactive flood risk maps
- Develop a flood risk index
- Generate engineering insights using Python and GIS
